## 介绍

这是我在学习 Vue 过程中做的一个 UI 框架，希望对你有用

## 开始使用

1. 添加 CSS 样式

使用本框架前， 请在 CSS 中开启 border-box .

```
*, *::before, *::after {box-sizing: border-box}
```

IE 8 及以上浏览器都支持此样式。

2. 安装 wheels

```
npm i --save vue-wheels-demo
```

3. 引入 wheels

```
import { Button } from "vue-wheels-demo";
import "vue-wheels-demo/dist/index.css";

export default {
  name: "App",
  components: {
    Button,
  },
}
```
